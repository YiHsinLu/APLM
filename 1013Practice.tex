\item
  p-value = 0.409
\item
  if p-value \textgreater{} \(\alpha\), then we couldn't reject \(H_0\).
  Hence we don't have enough information to say \(\beta_0=0\)
\item
  if p-value \textless{} \(\alpha\), then we reject \(H_0\).Hence
  \(\beta_0\neq0\)
\end{itemize}
